Here's the Celery flow:

  Files Used

  ┌─────────────────────────────────────────────────────────────────┐
  │  1. celery_worker.py  (entry point)                              │
  │     - Creates Celery app with Redis broker                       │
  │     - Imports all models (fixes SQLAlchemy relationships)        │
  │     - Imports tasks to register them                             │
  └─────────────────────────────────────────────────────────────────┘
                                ↓
  ┌─────────────────────────────────────────────────────────────────┐
  │  2. app/workers/tasks.py  (the tasks)                           │
  │     - schedule_interview_task:                                   │
  │       1. Gets candidate from DB                                  │
  │       2. Calls n8n webhook                                        │
  │       3. Updates interview_schedule table with gmeet_link       │
  └─────────────────────────────────────────────────────────────────┘
                                ↓
  ┌─────────────────────────────────────────────────────────────────┐
  │  3. app/routes/admin_routes.py  (FastAPI endpoint)              │
  │     - /admin/schedule-interview                                 │
  │     - Validates candidate                                        │
  │     - Finds available slot                                       │
  │     - Creates schedule record (status=pending)                   │
  │     - Calls schedule_interview_task.delay() ← queues to Redis   │
  └─────────────────────────────────────────────────────────────────┘
                                ↓
  ┌─────────────────────────────────────────────────────────────────┐
  │  Redis (message broker)                                         │
  │     - Stores task messages                                       │
  └─────────────────────────────────────────────────────────────────┘
                                ↓
  ┌─────────────────────────────────────────────────────────────────┐
  │  Celery Worker (running separately)                             │
  │     - Picks up task from Redis                                   │
  │     - Executes schedule_interview_task                          │
  │     - Updates DB with meet link                                  │
  └─────────────────────────────────────────────────────────────────┘
